## About The Project

<img src="assets/images/previews/preview.png" alt="Logo" width="100%" height="100%">

<p align="center">BetterDiscordPanel is a new panel that allows users to message and manage inside of their discord bots.</p>

**Current Features:**
* Messaging Interface with Discord Design
* Server Statistics
* Bot Statistics 
* Discord Light and Dark Theme

**Coming Soon:**
* Bot Profile Editing
* Electron Desktop Application
* Website with Documentation

## Getting Started

This project does not run on any servers and can be self hosted. Simply open 
```index.html``` to get started or use the control panel by opening ```BetterDiscordPanel.cmd``` and executing "W" for the web panel.

<!--
### Prerequisites

No prerequisites yet, however the Electron Desktop version will require some soon.

### Installation

1. Clone the repo
```sh
git clone https://github.com/D3VSJ/BetterDiscordPanel
```
**UPDATES only work if you have cloned the repository.**

2. Open ```login.html``` or open ```BetterDiscordPanel.cmd``` to run the program.

3. Login to your Discord Bot.

4. Start using BetterDiscordPanel!
